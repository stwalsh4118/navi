## Requirements

1. `up` or `k` moves cursor up
2. `down` or `j` moves cursor down
3. Cursor wraps at list boundaries
4. Cursor stays in bounds (0 to len(sessions)-1)

## Implementation Plan

Update `model.go` Update() method:
```go
case tea.KeyMsg:
    switch msg.String() {
    case "up", "k":
        if m.cursor > 0 {
            m.cursor--
        } else if len(m.sessions) > 0 {
            m.cursor = len(m.sessions) - 1 // wrap to bottom
        }
        return m, nil

    case "down", "j":
        if m.cursor < len(m.sessions)-1 {
            m.cursor++
        } else {
            m.cursor = 0 // wrap to top
        }
        return m, nil
    }
```
