## Implementation Plan

1. Implement `main()` function:
   ```go
   func main() {
       p := tea.NewProgram(initialModel(), tea.WithAltScreen())
       if _, err := p.Run(); err != nil {
           fmt.Fprintf(os.Stderr, "Error: %v\n", err)
           os.Exit(1)
       }
   }
   ```
2. Implement `initialModel()` function returning a new Model
3. Implement stub `Init()`, `Update()`, `View()` methods on Model
