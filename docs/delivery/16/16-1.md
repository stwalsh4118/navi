## Requirements

- Define `GitInfo` struct with fields for:
  - `Branch` (string): Current branch name
  - `Dirty` (bool): Whether there are uncommitted changes
  - `Ahead` (int): Commits ahead of remote
  - `Behind` (int): Commits behind remote
  - `LastCommit` (string): Short hash + subject of last commit
  - `Remote` (string): Remote URL (for GitHub detection)
- Define constants for git display:
  - Dirty indicator character (e.g., "‚óè")
  - Ahead/behind format strings
  - Git info separator
- Add JSON tags for serialization compatibility
