## Requirements

- Parse common GitHub remote URL formats:
  - `https://github.com/owner/repo.git`
  - `https://github.com/owner/repo`
  - `git@github.com:owner/repo.git`
  - `git@github.com:owner/repo`
  - `ssh://git@github.com/owner/repo.git`
- Extract owner and repo name
- Add fields to GitInfo or create separate GitHubInfo struct
- Return nil/empty for non-GitHub remotes
- Handle edge cases (no remote, private repos, etc.)

## Implementation Plan

1. Create `GitHubInfo` struct with Owner and Repo fields
2. Implement `parseGitHubRemote(remoteURL string) *GitHubInfo` function
3. Use regex or string parsing to handle different URL formats
4. Add GitHubInfo field to GitInfo struct (or parse on demand)
5. Create helper to construct GitHub URLs: `githubURL(info *GitHubInfo, path string) string`
