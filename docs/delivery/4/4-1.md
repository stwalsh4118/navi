## Requirements

Define styles for:
1. Status icon colors (yellow, green, magenta, cyan, red, dim)
2. Session name (bold)
3. Age/timestamp (right-aligned, normal)
4. Working directory (dimmed)
5. Message (dimmed, italic)
6. Selected row (highlighted)
7. Header/footer box borders

## Implementation Plan

Implement in `styles.go`:
```go
package main

import "github.com/charmbracelet/lipgloss"

var (
    // Status colors
    yellowStyle  = lipgloss.NewStyle().Foreground(lipgloss.Color("226"))
    greenStyle   = lipgloss.NewStyle().Foreground(lipgloss.Color("46"))
    magentaStyle = lipgloss.NewStyle().Foreground(lipgloss.Color("201"))
    cyanStyle    = lipgloss.NewStyle().Foreground(lipgloss.Color("51"))
    redStyle     = lipgloss.NewStyle().Foreground(lipgloss.Color("196"))
    dimStyle     = lipgloss.NewStyle().Foreground(lipgloss.Color("241"))

    // Text styles
    boldStyle    = lipgloss.NewStyle().Bold(true)
    italicStyle  = lipgloss.NewStyle().Italic(true)

    // Row styles
    selectedStyle = lipgloss.NewStyle().
        Background(lipgloss.Color("236")).
        Bold(true)

    // Box styles for header/footer
    boxStyle = lipgloss.NewStyle().
        Border(lipgloss.RoundedBorder()).
        BorderForeground(lipgloss.Color("240")).
        Padding(0, 1)
)
```
