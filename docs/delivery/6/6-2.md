## Requirements

1. Copy `hooks/notify.sh` to `~/.claude-sessions/hooks/`
2. Make the script executable
3. Handle missing source file gracefully
4. Print status message

## Implementation Plan

Add to `install.sh`:
```bash
# Get script directory
SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"

# Copy hook script
if [ -f "$SCRIPT_DIR/hooks/notify.sh" ]; then
    cp "$SCRIPT_DIR/hooks/notify.sh" "$HOOKS_DIR/"
    chmod +x "$HOOKS_DIR/notify.sh"
    echo "✓ Installed notify.sh hook"
else
    echo "✗ Error: hooks/notify.sh not found"
    exit 1
fi
```
